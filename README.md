# Headless Blog Frontend

A minimal, open-source blog frontend built with **Next.js**, **Markdown**, and **Tailwind CSS**.  
Designed to be fast, simple, and easy to extend with a headless CMS.

## Features
- Markdown-based blog posts
- Static site generation (SSG)
- Clean, readable typography
- SEO-friendly URLs
- CMS-ready architecture

## Tech Stack
- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Markdown (gray-matter, remark)

## Getting Started

Click **Use this template** on GitHub to create a new blog project
based on this repository.

Install dependencies and start the dev server:
```bash
npm install
npm run dev
